The Mobile Chatter Answers needs to know which Article Type and which fields need to be displayed. In this section, you can specify those articles types, the fields that will be shown, and in which order they should be shown. Note that the security settings around the visibility of Articles Types and Custom fields on these articles is unaffected by the settings of the mobile App. If an Article is not visible to a user, adding it to this section will not make it visible.
